# Medibot

![Medibot Logo](/images/logo.png)

## Overview

Medibot is a telemedicine web application designed to bridge the gap between patients and healthcare providers. Built using modern web technologies like **Next.js**, **Tailwind CSS**, and **Gemini API**, Medibot provides a seamless experience for remote medical consultations, prescriptions, and more.

## Features

- **Video Consultations**: Real-time video calls with doctors using WebRTC.
- **Appointment Booking**: Schedule appointments with doctors.
- **Prescription Management**: Digital prescriptions and medical records.
- **Chat Support**: Text-based communication with healthcare professionals.
- **Symptom Checker**: AI-powered symptom analysis using the Gemini API.
- **Responsive Design**: Mobile-friendly interface built with Tailwind CSS.
